# Introduction
This is my test package for calculator.

## Installation
```
npm i dntcalc --save
```

## Usage
Just follow the following steps to use this package.

```javascript
const calc= require('dntcalc')
const c1= new calc();

console.log(c1.Add(1,2));
console.log(c1.Multiply(2,2));

--Output
3
4
```
